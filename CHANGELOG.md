# Changelog

## 0.0.1

- Initial release.
- `codeTrace.copy` command — copies workspace-relative file path with optional line range to clipboard.
- Keybinding `Ctrl+Alt+C` / `Cmd+Alt+C`.
- Editor context menu entry under copy/paste group.
- `codeTrace.multiSelection` setting: `primary` (default), `perSelection`, `span`.
